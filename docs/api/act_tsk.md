# act_tsk — タスクの起動

## 種別
〔TI〕タスクコンテキスト・非タスクコンテキストの両方から呼出し可能

## C言語API
```c
ER ercd = act_tsk(ID tskid);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `tskid` | `ID` | 起動対象タスクのID番号 |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_ID` | `tskid` が不正・範囲外 |
| `E_CTX`〔s〕 | CPUロック状態で呼び出した |
| `E_NOEXS`〔D〕 | 対象タスクが未登録（動的生成対応カーネル） |
| `E_QOVR` | 起動要求キューイング数が上限を超えた |

## 機能
`tskid` で指定したタスクを起動する。対象タスクが休止状態（DORMANT）の場合は、
そのタスクを実行可能状態（READY）に遷移させる。対象タスクが休止状態でない場合は、
起動要求をキューイングする（キューイング数の上限を超えると `E_QOVR`）。

起動されたタスクは、初期優先度・拡張情報を持って実行開始する。

`TSK_SELF`（＝0）を指定すると自タスクを対象とする。

## 使用例
```c
void main_task(intptr_t exinf) {
    ER ercd = act_tsk(WORKER_TASK);
    if (ercd != E_OK) {
        syslog(LOG_ERROR, "act_tsk failed: %d", ercd);
    }
}
```
