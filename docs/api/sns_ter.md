# sns_ter — タスク終了禁止状態の参照

## 種別
〔TI〕タスクコンテキスト・非タスクコンテキストの両方から呼出し可能

## C言語API
```c
bool_t state = sns_ter(void);
bool_t state = isns_ter(void);   /* 非タスクコンテキスト用エイリアス */
```

## パラメータ
なし。

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `state` | `bool_t` | 終了禁止状態なら `true`、終了許可状態なら `false` |

## エラーコード
このサービスコールはエラーコードを返さない（`bool_t` を返す）。
CPUロック状態などの文脈からも呼び出せる。

## 機能
実行状態のタスク（自タスク）が終了禁止状態にあるか否かを参照し、`bool_t` で返す。
終了禁止状態（`dis_ter` 実行後で `ena_ter` 未実行）なら `true`、終了許可状態なら `false` を
返す。

実行状態のタスクが存在しない場合（非タスクコンテキストでアイドル中など）は `true` を返す。

終了禁止状態を変更できるのは自タスクのみであるため、排他制御なしに参照できる。
`isns_ter` は非タスクコンテキストから呼び出す場合のエイリアスである（実体は `sns_ter`）。

## 使用例
```c
void worker_task(EXINF exinf) {
    if (!sns_ter()) {
        dis_ter();          /* まだ終了禁止でなければ禁止する */
    }
    do_critical_section();
    ena_ter();
}
```

## 関連
- `dis_ter` — タスク終了の禁止
- `ena_ter` — タスク終了の許可
- `ras_ter` — タスクの終了要求
- 仕様書 4.3 タスク終了機能
