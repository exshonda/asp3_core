# ini_flg — イベントフラグの再初期化

## 種別
〔T〕タスクコンテキスト専用（非タスクから呼ぶと `E_CTX`）

## C言語API
```c
ER ercd = ini_flg(ID flgid);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `flgid` | `ID` | 対象イベントフラグのID番号 |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態 |
| `E_ID` | `flgid` が不正・範囲外 |

## 機能
`flgid` で指定したイベントフラグを再初期化する。

- イベントフラグの待ち行列につながれているタスクがあれば、それらの待ち状態を解除する。
  待ち解除されたタスクには、待ち状態に入ったサービスコールから `E_DLT` が返る。
- イベントフラグのビットパターンを、生成時（`CRE_FLG`）に指定した初期ビットパターン
  （`iflgptn`）に戻す。

待ち解除によって、より高い優先度のタスクが実行可能になった場合はディスパッチが行われる。

> 再初期化系サービスコール（`ini_*`）はオブジェクトを生成し直すものではないため、
> `E_OBJ`・`E_NOEXS` のような未生成・状態エラーは返さない（静的に生成済みのオブジェクト
> のみを対象とするため）。誤ったIDを指定した場合は `E_ID` となる。

## 使用例
```c
void recover_task(intptr_t exinf) {
    /* 異常検出後、イベントフラグを初期状態に戻す */
    ini_flg(SYS_FLG);
}
```
