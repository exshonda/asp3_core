# unl_cpu — CPUロック状態の解除

## 種別
〔TI〕タスクコンテキスト・非タスクコンテキストの両方から呼出し可能

## C言語API
```c
ER ercd = unl_cpu(void);
```

## パラメータ
なし

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |

> エラーは返らない。既にCPUロック解除状態であっても `E_OK` で正常終了する（ASP3仕様）。

## 機能
CPUロック状態を解除する。`loc_cpu` でCPUロック状態に遷移する前の状態
（割込み許可状態・ディスパッチ禁止状態であったか否か）に復帰させる。

ASP3では、**CPUロック状態でないときに `unl_cpu` を呼び出してもエラーにならず、
そのまま `E_OK` を返す**。`loc_cpu` と `unl_cpu` の対応は**ネストしない**
（μITRON4.0と異なり、ネストカウントを持たない）。CPUロック状態で `loc_cpu` を
何回呼んでいても、1回の `unl_cpu` でCPUロックは解除される。

CPUロック中はディスパッチが必要となるサービスコールを呼び出せないため、
`unl_cpu` による解除時にディスパッチャを明示的に起動する必要はない。

## 使用例
```c
void critical_section(intptr_t exinf) {
    loc_cpu();
    shared_counter++;
    unl_cpu();   /* CPUロック前の状態に復帰 */
}
```
