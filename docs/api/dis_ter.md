# dis_ter — タスク終了の禁止

## 種別
〔T〕タスクコンテキスト専用（非タスクコンテキストから呼ぶと `E_CTX`）

## C言語API
```c
ER ercd = dis_ter(void);
```

## パラメータ
なし。

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態で呼び出した |

## 機能
自タスクを終了禁止状態に遷移させる。終了禁止状態にあるタスクは、他タスクからの
`ras_ter`（タスク終了要求）によって即座には終了されず、終了要求はフラグとして
保留される。保留された終了要求は、`ena_ter` で終了許可状態に戻した時点で実行される。

クリティカルな処理の途中で強制終了されると不整合が生じる区間を保護するために用いる。
`dis_ter` と `ena_ter` で区間を囲む。

終了禁止状態は自タスクに対してのみ設定でき、対象タスクを指定する引数を持たない。

## 使用例
```c
void worker_task(EXINF exinf) {
    dis_ter();              /* この区間は終了要求を保留させる */
    do_critical_section();
    ena_ter();              /* 保留されていた終了要求があればここで終了する */
}
```

## 関連
- `ena_ter` — タスク終了の許可
- `sns_ter` — タスク終了禁止状態の参照
- `ras_ter` — タスクの終了要求
