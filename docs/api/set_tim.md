# set_tim — システム時刻の設定

## 種別
〔T〕タスクコンテキスト専用（非タスクコンテキストから呼び出すと `E_CTX`）

## C言語API
```c
ER ercd = set_tim(SYSTIM systim);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `systim` | `SYSTIM` | 設定するシステム時刻（`SYSTIM` = `uint64_t`、単位はマイクロ秒） |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態で呼び出した |

## 機能
カーネルが管理するシステム時刻を `systim` に設定する。

システム時刻は、カーネル内部でモノトニックに進む時刻（`monotonic_evttim`）に対する
オフセット（`systim_offset`）を更新することで設定される。すなわち、設定後のシステム時刻は
設定値を起点として進む。

`set_tim` はシステム時刻のオフセットのみを変更する。高分解能タイマのカウント値や、
登録済みのタイムイベント（周期通知・アラーム通知・タイムアウト等）の発生時刻には影響しない。
タイムイベントの発生は内部のモノトニックな時刻に基づくため、`set_tim` で時刻を変更しても
それらの相対的な発生タイミングは変わらない。

システム時刻を連続性を保ったまま少しずつ補正したい場合は、`adj_tim` を使う。

## 使用例
```c
void init_task(intptr_t exinf) {
    /* システム時刻を起点 0 に合わせ直す */
    set_tim(0U);
}
```
