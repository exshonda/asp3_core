# wai_sem — セマフォ資源の獲得

## 種別
〔T〕タスクコンテキスト専用（非タスクから呼ぶと `E_CTX`）

## C言語API
```c
ER ercd = wai_sem(ID semid);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `semid` | `ID` | 対象セマフォのID番号 |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了（資源を獲得した） |
| `E_ID` | `semid` が不正・範囲外 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態／ディスパッチ禁止状態 |
| `E_RLWAI` | 待ち状態が強制解除された（`rel_wai` / `ter_tsk`） |
| `E_DLT` | 待ち中にセマフォが削除された（動的生成対応カーネル） |

## 機能
`semid` で指定したセマフォから資源を1つ獲得する。セマフォの資源数が1以上の場合は、
資源数を1減じて即座にリターンする。資源数が0の場合は、自タスクをセマフォの待ち行列に
つなぎ、資源が返却される（`sig_sem`）まで待ち状態に遷移する。

ポーリングを行う場合は `pol_sem`、タイムアウトを設ける場合は `twai_sem` を使う。

## 使用例
```c
void worker(intptr_t exinf) {
    ER ercd = wai_sem(RESOURCE_SEM);
    if (ercd == E_OK) {
        /* クリティカルセクション */
        sig_sem(RESOURCE_SEM);
    }
}
```

## 関連
- `sig_sem` — セマフォ資源の返却
- `pol_sem` — 獲得（ポーリング、非タスクからも可）
- `twai_sem` — 獲得（タイムアウト付き）
