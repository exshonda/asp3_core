# ena_ter — タスク終了の許可

## 種別
〔T〕タスクコンテキスト専用（非タスクコンテキストから呼ぶと `E_CTX`）

## C言語API
```c
ER ercd = ena_ter(void);
```

## パラメータ
なし。

## リターンパラメータ
正常に終了の許可だけを行った場合は `E_OK` を返す。
保留されていた終了要求にしたがって自タスクが終了した場合は、このコールから戻らない。

| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード（終了した場合は戻らない） |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了（保留された終了要求がなかった） |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態で呼び出した |
| `E_SYS` | システムエラー（自タスク終了後に制御が戻った場合。正常時には返らない） |

## 機能
自タスクを終了許可状態に遷移させる（`dis_ter` による終了禁止状態を解除する）。

自タスクに終了要求が保留されている（`raster` がセットされている）状態で、かつ
ディスパッチが許可されている場合は、終了許可への遷移にともない、保留されていた
終了要求にしたがって自タスクをただちに終了させる。この場合、このコールから戻らない。

保留された終了要求がない場合は、終了許可状態に遷移して `E_OK` を返す。

終了許可状態は自タスクに対してのみ設定でき、対象タスクを指定する引数を持たない。

## 使用例
```c
void worker_task(EXINF exinf) {
    dis_ter();
    do_critical_section();
    ena_ter();              /* 保留された終了要求があればここで終了する */
    /* 終了要求がなければ続行 */
    do_other_work();
}
```
