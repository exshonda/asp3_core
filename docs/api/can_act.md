# can_act — タスク起動要求のキャンセル

## 種別
〔T〕タスクコンテキスト専用（非タスクコンテキストから呼ぶと `E_CTX`）

## C言語API
```c
ER_UINT ercd = can_act(ID tskid);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `tskid` | `ID` | 対象タスクのID番号。`TSK_SELF`（=0）で自タスク |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER_UINT` | 正常終了時はキャンセルした起動要求のキューイング数（0または1）、エラー時はエラーコード（負値） |

## エラーコード
| コード | 条件 |
|---|---|
| `0` または正の値 | 正常終了（キャンセルした起動要求のキューイング数） |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態で呼び出した |
| `E_ID` | `tskid` が不正・範囲外（`TSK_SELF` 以外を指定した場合のみチェック） |

## 機能
`tskid` で指定したタスクに対してキューイングされている起動要求をすべてキャンセルする。
キャンセルした起動要求のキューイング数を `ercd` に返す（ASP3では起動要求キューイング数の
上限は1のため、0または1）。起動要求がキューイングされていなかった場合は0を返す。

`TSK_SELF`（=0）を指定すると自タスクを対象とする。

戻り値の型が `ER_UINT` であり、正常終了時にも正の値を返しうる点に注意する。
エラー判定は値が負かどうかで行う。

## 使用例
```c
void main_task(EXINF exinf) {
    ER_UINT ercd = can_act(WORKER_TASK);
    if (ercd < 0) {
        syslog(LOG_ERROR, "can_act failed: %d", (int) ercd);
    } else {
        syslog(LOG_NOTICE, "cancelled %d activation request(s)", (int) ercd);
    }
}
```

## 関連
- `act_tsk` — タスクの起動（起動要求のキューイングを行う）
- `ref_tsk` — タスク状態の参照（`actcnt` で起動要求キューイング数を取得）
