# get_ipm — 割込み優先度マスクの参照

## 種別
〔T〕タスクコンテキスト専用

## C言語API
```c
ER ercd = get_ipm(PRI *p_intpri);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `p_intpri` | `PRI*` | 現在の割込み優先度マスクを格納する領域へのポインタ |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |
| `*p_intpri` | `PRI` | 現在の割込み優先度マスク（`TMIN_INTPRI` 〜 `TIPM_ENAALL`） |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した、またはCPUロック状態で呼び出した |

## 機能
現在の割込み優先度マスクの値を読み出し、`p_intpri` が指す領域に格納する。

全割込みが許可されている状態では `TIPM_ENAALL`（＝0）が格納される。それ以外では、
`chg_ipm` で設定した割込み優先度マスク（負値）が格納される。

タスクコンテキスト専用であり、CPUロック状態でも呼び出せない（いずれの場合も `E_CTX`）。

## 使用例
```c
void task(intptr_t exinf) {
    PRI ipm;
    if (get_ipm(&ipm) == E_OK) {
        syslog(LOG_NOTICE, "current IPM = %d", ipm);
    }
}
```

## 関連
- `chg_ipm` — 割込み優先度マスクの変更（対操作）
- `loc_cpu` / `unl_cpu` — CPUロック / 解除
