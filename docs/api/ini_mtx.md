# ini_mtx — ミューテックスの再初期化

## 種別
〔T〕タスクコンテキスト専用（非タスクから呼ぶと `E_CTX`／CPUロック状態でも `E_CTX`）

## C言語API
```c
ER ercd = ini_mtx(ID mtxid);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `mtxid` | `ID` | 対象ミューテックスのID番号 |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態（`CHECK_TSKCTX_UNL`） |
| `E_ID` | `mtxid` が不正・範囲外 |

## 機能
`mtxid` で指定したミューテックスを初期化する。

ミューテックスの待ち行列を初期化し、待っていたタスクの待ち状態を解除する。また、
ミューテックスがロックされていた場合はそのロックを解除する。ロックを解除した際、
そのミューテックスが優先度上限ミューテックス（`TA_CEILING`）であれば、ロックして
いたタスクの現在優先度を再計算し、必要に応じて引き下げる（優先度上昇状態の解消）。
これらの処理によりディスパッチが必要になればディスパッチが発生する。

なお、待ち解除されたタスクには `ini_mtx` 固有のエラーコードは設定されず、待ち行列の
初期化に伴い `init_wait_queue` 経由で待ちが解除される。

## 使用例
```c
void recovery(intptr_t exinf) {
    /* 異常状態からの復旧時にミューテックスを既知の状態へ戻す */
    ini_mtx(SHARED_MTX);
}
```
