# get_inf — 自タスクの拡張情報の参照

## 種別
〔T〕タスクコンテキスト専用（非タスクコンテキストから呼ぶと `E_CTX`）

## C言語API
```c
ER ercd = get_inf(EXINF *p_exinf);
```

## パラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `p_exinf` | `EXINF *` | 拡張情報を格納する領域へのポインタ（`EXINF` は `intptr_t`） |

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `ercd` | `ER` | 正常終了（`E_OK`）またはエラーコード |
| `*p_exinf` | `EXINF` | 自タスクの拡張情報 |

## エラーコード
| コード | 条件 |
|---|---|
| `E_OK` | 正常終了 |
| `E_CTX` | 非タスクコンテキストから呼び出した／CPUロック状態で呼び出した |

## 機能
自タスクの拡張情報（タスク生成時に指定した `exinf`）を読み出し、`p_exinf` が指す領域に
格納する。拡張情報は、同一のタスクエントリ関数を複数のタスクで共有する場合などに、
タスクごとに異なるデータを渡す目的で利用できる。

このサービスコールは常に自タスクを対象とし、対象タスクを指定する引数を持たない。
タスクのエントリ関数は起動時に拡張情報を引数で受け取るが、`get_inf` を使えば
エントリ関数の引数を保持していなくても後から拡張情報を取得できる。

## 使用例
```c
void worker_task(EXINF exinf) {
    EXINF inf;
    get_inf(&inf);          /* 引数 exinf と同じ値が得られる */
    process((void *) inf);
}
```
