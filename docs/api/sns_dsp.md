# sns_dsp — ディスパッチ禁止状態の参照

## 種別
〔TI〕タスクコンテキスト・非タスクコンテキストの両方から呼出し可能（CPUロック状態でも呼出し可能）

## C言語API
```c
bool_t state = sns_dsp(void);
```

## パラメータ
なし

## リターンパラメータ
| 名称 | 型 | 説明 |
|---|---|---|
| `state` | `bool_t` | ディスパッチ禁止状態なら `true`、ディスパッチ許可状態なら `false` |

## エラーコード
なし。

> この関数は `bool_t`（真偽値）を返し、`ER` 型のエラーコードを返さない。
> どのようなコンテキスト・状態（CPUロック状態を含む）からでも呼び出せ、エラーにならない。

## 機能
処理単位がディスパッチ禁止状態か否かを参照する。ディスパッチ禁止状態であれば
`true`、ディスパッチ許可状態であれば `false` を返す。実装上は内部変数 `enadsp`
（ディスパッチ許可フラグ）の否定を返す。

状態を参照するだけで、カーネルの状態を変化させない。CPUロック状態であっても
呼び出すことができ、エラーにならない。

## 使用例
```c
void conditional_dispatch_enable(void) {
    if (sns_dsp()) {
        ena_dsp();   /* ディスパッチ禁止中なら許可に戻す */
    }
}
```

## 関連
- `dis_dsp` / `ena_dsp` — ディスパッチの禁止 / 許可
- `sns_dpn` — ディスパッチ保留状態の参照
- `sns_loc` — CPUロック状態の参照
- 仕様書 4.7 システム状態管理機能
